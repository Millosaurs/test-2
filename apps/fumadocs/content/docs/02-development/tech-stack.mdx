---
title: Tech Stack
description: Libraries and tools used
---

- Monorepo: Turborepo, Bun
- Frontend: Next.js (App Router), React, TypeScript, TailwindCSS, shadcn/ui, lucide-react
- Docs: Fumadocs (MDX) on Next.js
- Backend: Next.js route handlers, oRPC, Better Auth, Drizzle ORM, PostgreSQL
- ORM/DB: Drizzle + Postgres
- Payments: Lemon Squeezy (hosted checkout; `@lemonsqueezy/lemonsqueezy.js`)
- Media: Cloudinary (admin image upload/delete)
- Tooling: TypeScript, Turbo tasks
